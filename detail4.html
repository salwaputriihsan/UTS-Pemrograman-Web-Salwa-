Beranda / Flutter Tailor Shop

Cerita Singkat

Project ini merupakan tugas UTS mata kuliah Pemrograman Multiplatform. Dalam tugas ini kami diminta membuat tampilan aplikasi menggunakan Flutter di Visual Studio Code.

Halaman yang dibuat terdiri dari splashscreen, halaman utama, register, login, dan dashboard. Kami juga diminta membuat tampilan yang modern dan menarik agar terlihat seperti aplikasi asli.

Pengalaman Saat Mengerjakan

Jujur saat mengerjakan project ini rasanya cukup sulit, karena harus banyak coding untuk mengatur layout, warna, dan tampilan agar terlihat rapi.

Saya belajar banyak tentang Flutter, terutama bagaimana mengatur widget, navigasi halaman, dan membuat desain web/aplikasi menjadi lebih aesthetic.

Meskipun cukup melelahkan, saya merasa senang ketika hasil akhirnya berhasil menjadi tampilan web yang cantik dan sesuai dengan konsep yang saya inginkan.
